# Ticket-Zetu-API

# **Ticket-Zetu-API - Quick Setup**

1. **Clone the repo:**

   ```bash
   git clone https://github.com/chickens-s/ticket-zetu-api.git
   cd ticket-zetu-api
   ```

2. **Install Reflex (for hot-reloading):**

   ```bash
   make install-reflex
   ```

3. **Create `.env` file:**

   Add your environment variables (e.g., `PORT`, `DB_NAME`, `CLOUDINARY_API_KEY`) in the `.env` file.

4. **Run the app:**

   ```bash
   make run
   ```

   Or, for development with hot-reload:

   ```bash
   make dev
   ```

5. **Build the app:**

   ```bash
   make build
   ```

6. **Test the app:**

   ```bash
   make test
   ```
